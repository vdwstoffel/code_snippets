# Navigation

```bash
pwd                 # Check entire working path
ls                  # List items in directory
cd path/to/foldfer  # change directory
cd ..               # go back one
cd ~                # go to root
```

# Zip

```bash
tar -xvzf /path/to/yourfile.tgz
```
